# 🌿 BENICIA Ambientaciones — Sitio Web

Sitio web profesional desarrollado para **BENICIA Ambientaciones**, empresa dedicada a la ambientación de bodas y eventos en Buenos Aires, Argentina.

> 🔗 **[Ver sitio en vivo →](#)** *(reemplazar con URL real)*

---

## 📋 Sobre el proyecto

BENICIA necesitaba una presencia web que transmitiera la elegancia de su marca y, al mismo tiempo, funcionara como herramienta de captación de clientes. Desarrollé un sitio completo que incluye:

- **Landing page** con diseño editorial y estética premium
- **Formulario de captación de leads** integrado con Google Forms
- **Galería de eventos** con grid responsivo y animaciones
- **Sección de testimonios** de clientes reales
- **Diseño 100% responsive** optimizado para móviles

### Resultado de negocio
La implementación del sitio con su sistema de captación de leads **contribuyó a generar más de 30 ventas cerradas** para la empresa.

---

## 🛠️ Tecnologías

| Tecnología | Uso |
|---|---|
| **HTML5** | Estructura semántica |
| **CSS3** | Design system con variables CSS, grid, flexbox, animaciones |
| **JavaScript** | Interactividad, IntersectionObserver, navegación mobile |
| **Google Fonts** | Tipografías: Cormorant Garamond, Great Vibes, Inter |
| **Google Forms** | Integración de formulario de contacto para captación de leads |

---

## ✨ Características técnicas

- **Design System propio** con variables CSS para colores, tipografías y transiciones
- **Animaciones on-scroll** usando IntersectionObserver (reveal, stagger)
- **Navbar adaptativa** que cambia de estilo transparente a sólido al hacer scroll
- **Menú hamburguesa** con panel lateral animado para mobile
- **Galería con grid CSS** y clases dinámicas (vertical, horizontal, grande)
- **Performance optimizada** con event listeners pasivos y lazy animations
- **Scrollbar personalizado** acorde a la estética de la marca
